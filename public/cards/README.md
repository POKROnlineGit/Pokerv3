# Card Images

This directory should contain 52 playing card images, one for each card in a standard deck.

## Naming Convention

Each card image must be named exactly as follows:

### Format: `{Rank}{Suit}.png`

- **Rank**: `2`, `3`, `4`, `5`, `6`, `7`, `8`, `9`, `T` (for 10), `J` (Jack), `Q` (Queen), `K` (King), `A` (Ace)
- **Suit**: `h` (hearts), `d` (diamonds), `c` (clubs), `s` (spades) - **all lowercase**
- **Extension**: `.png`

### Examples

- `Ah.png` = Ace of Hearts
- `Tc.png` = Ten of Clubs
- `5d.png` = Five of Diamonds
- `Ks.png` = King of Spades

**Important**: All suits must be lowercase (`h`, `d`, `c`, `s`), not uppercase. The rank `T` represents the number 10.

## Image Requirements

- Format: PNG (with transparency recommended)
- Recommended size: At least 200x280 pixels
- Aspect ratio: Approximately 5:7 (width:height)
